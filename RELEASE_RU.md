# История изменений — luci-theme-argondash

## v1.0.1 [ 2026.04.28 ]

Форк [luci-theme-argon](https://github.com/jerrykuku/luci-theme-argon) (jerrykuku).
Полный визуальный рефакторинг в стиле [Argon Dashboard 2](https://www.creative-tim.com/product/argon-dashboard) (Creative Tim).

### Визуальная часть
- **Топбар**: эффект матового стекла (`backdrop-filter: blur`) на белой полосе со скруглёнными нижними углами; `position: sticky` — остаётся виден при прокрутке
- **Сайдбар**: тёмный градиентный фон (`#42424a → #191919`), активный пункт выделен pill-кнопкой, скруглённый правый край, плавные hover-переходы
- **Акцентная полоса**: тонкий цветной разделитель (`#4F62BC`, `0.5rem`) между топбаром и карточками контента с равными вертикальными отступами
- **Карточки**: `border-radius: 0.75rem`, мягкая тень, белый фон
- **Страница входа**: glassmorphism-карточка поверх полноэкранного фона (изображение / видео / Bing / Wallhaven / Unsplash), эффект размытия

### Исправления безопасности (относительно upstream argon)
- **Защита от CSS-инъекций**: все UCI-значения (цвета, числа, режимы) проверяются регулярными выражениями до вставки в блок `<style>` (`san_color`, `san_num`, `san_mode`)
- **XSS через URL фона**: `bgUrl` и `bgMime` экранируются через `entityencode()` во всех HTML-атрибутах
- **Path traversal в скрипте обоев**: `WEB_PIC_SRC` проверяется по строгому whitelist; суффикс очищается до `[A-Za-z0-9-]`
- **URL-инъекция в API-запросы**: `tag_id` / `collection_id` санитизированы; `EXACT_RESO` проверяется через `case` вместо `[ -eq ]`

### Сборка и инфраструктура
- **POSIX-совместимость shell**: заменены bash-специфичные `[[ ]]` на `case` в GitHub Actions workflow
- **Исправлен push-триггер**: скорректирован `push.paths` — теперь указывает на корень репозитория
- **Матричная сборка**: `.ipk` для OpenWrt 23.05 / 24.10 и `.apk` для 25.12 / snapshot
- **Основа для i18n**: добавлен `po/templates/luci-theme-argondash.pot` с 13 переводимыми строками; в CI активирован `CONFIG_LUCI_LANG_ru=y`
- **Манифест**: `manifest.json` обновлён — только реально существующие иконки, современный PWA-формат (`purpose`, `display`, `theme_color`)
- **Удалён мёртвый код**: `sysauth.js` (недостижимый JS-вид) удалён

### Совместимость
- Проверена совместимость с LuCI API для OpenWrt **23.05**, **24.10**, **25.12**
- Все template-глобальные переменные (`dispatched`, `version.*`, `getspnam`, `lua_active` и др.) подтверждены во всех трёх ветках
- LESS-исходники ресинхронизированы с компилированным CSS; все CSS-переменные совпадают
